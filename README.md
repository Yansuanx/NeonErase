### Overview
**PurifyNeon** is a lightweight, web-based image purification utility designed for rapid watermark and noise removal. Built with a cyberpunk aesthetic, it leverages advanced image inpainting algorithms to restore visual clarity to media streams with minimal latency.

---

### Core Capabilities
* **Inpainting Engine:** Powered by OpenCV’s Telea algorithm, capable of detecting and seamlessly filling regions based on brightness thresholds.
* **Web-First Interface:** A responsive, sleek dashboard featuring glassmorphism design, built for cross-platform efficiency (Mobile & Desktop).
* **Adaptive Processing:** Real-time parameter adjustment for restoration depth, giving you total control over the "purge" intensity.

### Quick Start

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/yourusername/PurifyNeon.git](https://github.com/yourusername/PurifyNeon.git)
    cd PurifyNeon
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Deploy the engine:**
    ```bash
    python app.py
    ```

4.  **Access the Console:**
    Open your browser and navigate to `http://127.0.0.1:5000` to mount your media.

### Tech Stack
* **Backend:** Flask (Python Web Server)
