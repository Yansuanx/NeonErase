# ⚡ NeonErase — AI 水印精准去除工具

> **赛博朋克紫色主题 · 纯前端 · 本地处理 · 隐私安全**

---

## 软件名称

**NeonErase**（霓虹抹除）— 灵感来自赛博朋克霓虹美学，主色调为炫丽紫色，界面风格类似 Gemini 手机版的流畅卡片设计。

## 使用方法

### 方式一：直接打开（推荐）
1. 解压 `NeonErase.zip`
2. 双击 `index.html` 用浏览器打开
3. 即可使用全部功能

### 方式二：本地服务器（视频功能需要）
```bash
# Python 3
cd NeonErase
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```

## 功能说明

### 图片水印去除
| 模式 | 说明 |
|------|------|
| 🤖 自动检测 | 自动识别半透明像素和均匀色块水印 |
| ✏️ 手动标记 | 在图片上拖拽框选水印区域，支持多区域 |
| 🎨 颜色识别 | 指定目标颜色和容差，精准去除同色水印 |
| 🔮 深度修复 | 多次迭代修复，适合复杂背景 |
| 📡 频域分析 | 频域特征分析去除规律性水印 |
| 🔬 边缘增强 | 边缘保留修复，适合细节丰富图片 |

**参数说明：**
- **去除强度**：修复像素与周围背景的混合比例（10–100）
- **边缘羽化**：水印边缘过渡平滑度（0–30）
- **修复迭代**：修复算法迭代次数，越高越精细（1–5）
- **颜色容差**：颜色识别模式下的匹配范围（5–120）

**预览功能：**
- 处理完成后显示前后对比，拖动中间滑块查看效果
- 支持多区域标记，标记后可清除重来

### 视频水印去除
| 模式 | 说明 |
|------|------|
| 🤖 自动检测 | 自动检测视频角落常见水印位置 |
| ✏️ 手动标记 | 在视频预览上框选水印区域 |
| 🎨 颜色识别 | 按颜色去除视频水印 |
| ⏱ 时序分析 | 利用帧间信息去除动态水印 |
| 📌 静态水印 | 优化处理固定位置水印 |
| 💫 动态水印 | 处理移动/变化水印 |

**输出格式：** WebM（VP9）/ MP4，支持保留原始音轨

## 技术特性

- **纯前端处理**：所有图像/视频处理均在浏览器本地完成，文件不上传任何服务器
- **Canvas API**：基于 HTML5 Canvas 的像素级图像处理
- **MediaRecorder API**：视频帧处理与重新编码
- **修复算法**：基于加权邻域平均的 Exemplar-based Inpainting
- **形态学操作**：膨胀算法扩展水印掩码边界
- **响应式设计**：完美适配手机、平板、桌面
